# APEX SYSTEMS - Enterprise ERP System 🚀

نظام إدارة موارد المؤسسات (ERP) متكامل، مصمم بواجهة عصرية واحترافية باستخدام أحدث تقنيات الويب.

## 🌟 المميزات الرئيسية
- **لوحة تحكم ذكية**: عرض إحصائيات حية ورسوم بيانية (Chart.js).
- **الإدارة المالية**: متابعة الفواتير، الإيرادات، والمصروفات.
- **إدارة الموارد البشرية (HR)**: إدارة الموظفين، الرواتب، الحضور، والإجازات.
- **إدارة المخازن والمبيعات**: تتبع المخزون والطلبات والعملاء.
- **دعم الوضع الليلي**: واجهة مريحة للعين مع إمكانية التبديل.
- **تصميم متجاوب**: يعمل بكفاءة على جميع الأجهزة.

## 🛠️ التقنيات المستخدمة
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Backend**: PHP (API Architecture).
- **Database**: SQLite (No-setup required).
- **Icons & Charts**: FontAwesome 6, Chart.js.

## 🚀 كيفية التشغيل محلياً
1. قم بتحميل المشروع ووضعه داخل مجلد `htdocs` في XAMPP.
2. تأكد من تفعيل خدمة **Apache**.
3. افتح المتصفح وانتقل إلى: `http://localhost/nexus_erp/`

## 📁 هيكل المشروع
- `index.html`: الصفحة الرئيسية (SPA).
- `style.css`: التصميم والهوية البصرية.
- `script.js`: المنطق البرمجي والربط مع الـ API.
- `api/`: يحتوي على ملفات الـ Backend وقاعدة البيانات.
